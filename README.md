## 阶段性项目目标

### Java + AI 项目

计划完成一个基于 Spring Boot 的 AI 应用项目，例如：

- AI 简历优化助手
- AI 学习规划助手
- AI 知识库问答系统
- AI 面试助手

项目目标：

- 使用 Spring Boot 搭建后端服务
- 接入大模型 API
- 实现用户输入、模型回复、历史记录保存
- 支持流式输出
- 支持文档上传与知识库问答
- 使用 MySQL 保存用户记录
- 整理完整 README、接口说明和运行截图

### Python + AI 项目

计划完成一个基于 Python 的 AI 应用项目，例如：

- 本地文档知识库问答系统
- AI 论文阅读助手
- AI 日报总结工具
- AI 代码解释助手

项目目标：

- 使用 Python 完成核心 AI 逻辑
- 使用 FastAPI 或 Streamlit 构建应用
- 实现文档解析、文本切分、Embedding 和向量检索
- 实现 RAG 问答流程
- 支持多轮对话或历史记录
- 整理完整 README、运行方式和效果展示
